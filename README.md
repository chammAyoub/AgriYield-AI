# 🌱 AgriYield AI

**Système intelligent d'irrigation et d'anticipation des carences nutritives.**

Contrairement aux systèmes classiques basés sur la simple détection, **notre IA est strictement prédictive**. Elle n'attend pas que la plante souffre : en analysant simultanément 15 paramètres agronomiques complexes (pH, apports, climat), notre modèle Machine Learning **anticipe** et **prédit** les carences nutritives bien avant l'apparition des premiers symptômes visuels.

## 🚀 Fonctionnalités Clés
* **💧 Irrigation Intelligente (IoT) :** Suivi du débit d'eau en temps réel et identification immédiate des fuites et sur-irrigations.
* **🧠 IA Prédictive :** Diagnostic agronomique préventif grâce à un modèle ML (Scikit-learn / XGBoost).
* **☁️ Cerveau Cloud :** Backend robuste (FastAPI) pour le filtrage anti-bruit et l'orchestration des flux de données.
* **📱 Réactivité Immédiate :** Application mobile B2B (React Native) avec alertes vibratoires inarrêtables en cas d'urgence hydrique.

## 🛠️ Stack Technique
* **Couche Matérielle :** Arduino Uno, Capteur de débit YF-S201, Module SIM800L
* **Backend & API :** Python, FastAPI, SQLite
* **Intelligence Artificielle :** Scikit-learn, XGBoost, Pandas
* **Frontend Mobile :** React Native, Expo
